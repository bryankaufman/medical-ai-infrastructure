# Medical AI Infrastructure

Development infrastructure and MCP orchestration for medical AI including 49-service orchestrator, knowledge base, and automation.

## Features

- **49-Service Orchestrator**: Unified MCP service management
- **Knowledge Base**: GraphRAG-powered medical knowledge
- **Port Management**: Intelligent port allocation
- **Automation**: CI/CD and workflow automation

## Architecture

- **Orchestrator**: Central service coordinator
- **Memory System**: Persistent context and knowledge
- **Port Manager**: System-wide port registry
- **MCP Servers**: Healthcare-specific protocol servers

## Installation

```bash
# Clone and setup
git clone https://github.com/bryankaufman/medical-ai-infrastructure.git
cd medical-ai-infrastructure

# Install dependencies
pip install -r requirements.txt
```
